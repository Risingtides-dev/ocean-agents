# Credits

Ocean Agents is built by people who believe powerful agent behavior should be
inspectable, portable, and useful across providers and interfaces.

## Ocean contributors

Thank you to the Ocean maintainers, reviewers, operators, and early testers who
turned surface-specific interaction knowledge into reusable authored profiles,
kept deterministic operations in narrow harnesses, challenged unsafe automation,
and separated public package mechanisms from private deployment knowledge.

The Bonzai worktree-gardening package, profile composer, manifest router,
file-courier reference, Slack transport adapter, and repository documentation
are original Ocean/Rising Tides project work. The public repository begins as a
clean package snapshot rather than importing private operational history.

## Foundations we rely on

- **Python contributors and the Python Software Foundation** — for the standard
  library used by every executable Python path in this repository.
- **Git contributors** — for the version-control primitives Bonzai inspects and
  invokes conservatively.
- **Slack platform and API teams** — for the documented Web API used by the
  optional generic file-courier transport. No Slack SDK is bundled.
- **GitHub and GitHub Actions maintainers** — for public hosting and the
  `actions/checkout` and `actions/setup-python` workflow building blocks used in
  CI.
- **Ocean OS and Ocean Surface contributors** — for the runtime, daemon, and
  client contracts these reusable packages target.
